# البدائل — دمبل وماكينة لكل تمرين أساسي

إذا الجهاز مشغول أو مش متوفر بالنادي، استخدم البديل بنفس الـ sets × reps ونفس الراحة. البديل يستهدف نفس العضلة بنفس النمط الحركي — سجّل وزن البديل منفصل لأنه ما بينقارن بوزن التمرين الأساسي.

**اليوم ١ – دفع (صدر/كتف/تراي)**

| التمرين الأساسي | بديل دمبل | بديل ماكينة/كيبل |
|---|---|---|
| Bench Press | Flat Dumbbell Press *(نفس الحركة، مدى أعمق)* | Chest Press Machine أو Smith Bench Press |
| Shoulder Press | Seated Dumbbell Shoulder Press | Shoulder Press Machine أو Smith OHP |
| Lateral Raise | *(هو أصلاً دمبل)* — Leaning Single-arm Lateral Raise | Cable Lateral Raise *(توتر ثابت — ممتاز)* أو Lateral Raise Machine |
| Triceps Pushdown | Overhead Dumbbell Extension أو Dumbbell Kickback | Triceps Extension Machine أو Rope Pushdown |

**اليوم ٢ – سحب (ظهر/باي)**

| التمرين الأساسي | بديل دمبل | بديل ماكينة/كيبل |
|---|---|---|
| Pull-ups / Lat Pulldown | Dumbbell Pullover *(لاتس بمدى طويل)* | Assisted Pull-up Machine أو Plate-loaded Pulldown |
| Chest-Supported Row | Chest-Supported DB Row على بنش مائل *(الأقرب للأساسي)* | Seated Cable Row أو Machine Row (Hammer Strength) |
| Single-arm Lat Pulldown | Single-arm Dumbbell Row *(الكوع باتجاه الحوض، نفس القاعدة)* | Machine Pullover أو Single-arm Seated Row |
| Rear Delt Fly | Bent-over Reverse Fly بالدمبل | Reverse Pec Deck *(أسهل ضبط)* أو Cable Rear Delt Fly |
| Biceps Curl | Alternating DB Curl أو Incline DB Curl *(مدّ أعمق)* | Cable Curl بالبار أو Preacher Curl Machine |

**اليوم ٣ – رجل**

| التمرين الأساسي | بديل دمبل | بديل ماكينة/كيبل |
|---|---|---|
| Squat | Goblet Squat أو DB Bulgarian Split Squat | Hack Squat أو Smith Squat |
| Romanian Deadlift | Dumbbell RDL *(نفس الحركة تماماً)* | Lying/Seated Leg Curl أو 45° Back Extension |
| Leg Press | DB Walking Lunges أو Goblet Squat | Hack Squat أو V-Squat Machine |
| Calf Raise | Single-leg DB Calf Raise على درجة | Seated Calf Machine أو Calf Press على Leg Press |

**اليوم ٤ – علوي كامل**

| التمرين الأساسي | بديل دمبل | بديل ماكينة/كيبل |
|---|---|---|
| Incline Press | Incline Dumbbell Press *(30-45°)* | Incline Chest Press Machine أو Smith Incline |
| Lat Pulldown Wide Grip | Dumbbell Pullover | Assisted Pull-up Machine *(قبضة واسعة)* |
| Face Pull | Bent-over Reverse Fly مع سحبة للوجه | Reverse Pec Deck *(قبضة محايدة)* |
| Lateral Raise | Leaning Single-arm Lateral Raise | Cable أو Machine Lateral Raise |
| Hammer Curl | *(هو أصلاً دمبل)* — Cross-body Hammer Curl | Rope Hammer Curl على الكيبل |

**قواعد اختيار البديل:**

* **الأولوية:** التمرين الأساسي ← بديل الماكينة/الكيبل ← بديل الدمبل — الماكينة أسهل بضبط الحمل التدريجي
* البديل مؤقت: إذا صرت تستخدمه أكثر من الأساسي لمدة أسبوعين+، اعتبره الأساسي الجديد وسجّل أوزانه
* بالتمارين المركبة (squat, bench, RDL) البديل بالدمبل رح يكون بوزن أقل بكثير — طبيعي، ركّز على نفس الـ reps والإحساس بالعضلة
* نفس قواعد البرنامج: نزول بطيء، آخر تكرار صعب (1-2 بالخزان)، وزيد 2.5 كيلو أو تكرار كل أسبوع
